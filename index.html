Comment ça marche ?

Simple pour le client, pratique pour toi.

📲

1. Le client scanne

Il ouvre ton site GitHub Pages avec un lien ou un QR code.

🛒

2. Il commande

Il ajoute ses produits, choisit ses sauces et valide.

👨‍🍳

3. Tu prépares

La commande apparaît dans l’espace admin du navigateur.

Important : cette version GitHub Pages fonctionne sans PHP. Les commandes sont sauvegardées dans le navigateur de l’appareil qui les reçoit. Pour recevoir les commandes de tous les clients sur ton téléphone en temps réel, il faudra ensuite connecter Google Forms, Google Sheets ou Firebase.
