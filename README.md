# 🍽️ Lerisfood

Lerisfood est un petit site de commande en ligne pour une activité de vente de nourriture.

Le client peut consulter le menu, ajouter des produits au panier, choisir ses sauces, laisser une note, puis valider sa commande.  
Le paiement se fait sur place.

## 🚀 Fonctionnalités

- Page d’accueil
- Menu avec produits
- Recherche dans le menu
- Panier dynamique
- Quantités modifiables
- Choix des sauces
- Note client
- Validation de commande
- Espace admin
- Statuts de commande :
  - À préparer
  - Prêt
  - Terminé
- Suppression de commandes
- Export CSV des commandes
- Sauvegarde locale dans le navigateur
