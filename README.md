# Lerisfood — Version GitHub Pages

Cette version est compatible GitHub Pages car elle utilise uniquement :

- HTML
- CSS
- JavaScript

Aucun PHP nécessaire.

## Fichiers

- `index.html`
- `assets/style.css`
- `assets/app.js`

## Fonctionnalités

- Accueil
- Menu
- Panier dynamique
- Sauces et notes
- Validation commande
- Espace admin avec code `1234`
- Statuts : À préparer / Prêt / Terminé
- Export CSV
- Sauvegarde locale dans le navigateur

## Important

GitHub Pages ne peut pas faire tourner PHP.
Cette version sauvegarde les commandes dans le navigateur avec `localStorage`.

Pour recevoir les commandes de tous les clients sur ton téléphone en direct,
il faudra connecter plus tard Google Forms, Google Sheets, Firebase ou un backend.
